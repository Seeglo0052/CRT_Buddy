## ?? 技术细节

### 布局结构
```python
QHBoxLayout (主布局)
├── QWidget (左侧 - 固定220px)
│   └── CRT屏幕 + 吉祥物 (绘制)
│
└── QVBoxLayout (右侧)
    ├── 状态栏
    ├── 输入框
    ├── 按钮1 (GENERATE)
    ├── 按钮2 (IMAGE)
    ├── 按钮3 (RANDOM)
    └── 电源按钮 (OFF)
```

### 按钮样式
```css
/* 长条按钮 - 横向渐变 */
background: qlineargradient(x1:0, y1:0, x2:1, y2:0,
  stop:0 灰色,
  stop:0.4 彩色1,
  stop:0.6 彩色2,
  stop:1 灰色);

/* 圆形按钮 - 径向渐变 */
background: qradialgradient(
  cx:0.5, cy:0.5, radius:0.8,
  stop:0 亮灰,
  stop:0.5 中灰,
  stop:1 暗灰);
```

## ?? 使用方法

### 启动
```bash
python main.py
```

### 功能
1. **移动窗口** - 拖动任意位置
2. **输入文字** - 右侧输入框
3. **生成Meme**:
   - GENERATE - 生成文字Meme
   - IMAGE - 上传图片
   - RANDOM - 随机生成
4. **关闭程序** - 点击OFF按钮

### 眼睛跟踪
- 移动鼠标到屏幕任意位置
- 左侧吉祥物眼睛会跟随
