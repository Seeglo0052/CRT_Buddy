Quick Start

1) Install dependencies (preferably in a `.venv`):
   - pip install PyQt6 Pillow numpy opencv-python pygame pywin32 requests

2) Run (pick one):
   - python start.py    # runs a quick dependency self-check
   - python CRT_Buddy.py
   - run or double-click run_app.bat  # one-click on Windows

3) Use it:
   - Text → type a phrase, click GENERATE MEME → check output/
   - Image → click UPLOAD IMAGE, apply effects → save to output/
   - Random → click RANDOM Y2K EFFECT → save

Troubleshooting:
- If you get ModuleNotFoundError (e.g., PyQt6), ensure VS Code uses the workspace `.venv` and install the deps there.
- If the window doesn’t show up or freezes, try run_app.bat or reinstall PyQt6.
