## ?? 技术细节

### 眼睛跟踪算法
```python
# 计算鼠标到吉祥物的方向
dx = mouse_x - mascot_x
dy = mouse_y - mascot_y
distance = sqrt(dx? + dy?)

# 归一化并缩放
eye_offset_x = (dx / distance) * 8
eye_offset_y = (dy / distance) * 8
```

### 动画系统
- **主计时器**: 50ms（20 FPS）
- **扫描线**: 每帧移动3像素
- **眨眼**: 60-100帧触发一次
- **LED呼吸**: sin波计算亮度

### 渲染层次
1. CRT外壳（最底层）
2. 屏幕玻璃
3. 吉祥物角色
4. 扫描线效果
5. 电源LED（最顶层）
