你的Y2K风格桌面宠物 & Meme生成器

---

## ?? 下载

**Windows 64位**
- 文件名: `CRT_Buddy.exe`
- 大小: 21 MB
- 系统要求: Windows 10/11 (64位)

---

## ?? 快速开始

### 1. 运行程序
```
双击 CRT_Buddy.exe
```

### 2. 开始创作
- **生成文字Meme**: 输入文字 → 点击 [? GENERATE MEME]
- **处理图片**: 拖拽图片到窗口 → 自动添加Y2K特效
- **随机生成**: 点击 [? RANDOM Y2K EFFECT]

### 3. 查看输出
生成的文件保存在 `output/` 文件夹

## ?? 常见问题

### Q: Windows提示"保护了你的电脑"？
**A**: 点击"更多信息" → "仍要运行"（正常现象）

### Q: 找不到生成的图片？
**A**: 查看程序所在目录的 `output` 文件夹

### Q: 提示缺少DLL？
**A**: 安装 [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)
