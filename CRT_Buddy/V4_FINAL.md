## ?? 核心特性

### 1. 全屏幕眼睛跟踪
```python
# 获取全局鼠标位置
self.global_mouse_pos = QCursor.pos()

# 计算窗口屏幕坐标
window_pos = self.mapToGlobal(QPoint(0, 0))

# 计算眼睛方向
dx = global_mouse_x - mascot_screen_x
dy = global_mouse_y - mascot_screen_y
```

**效果**: 不管鼠标在屏幕哪里，吉祥物都会看向它！

### 2. 金属Y2K按钮
```css
/* 金属渐变效果 */
background: qlineargradient(
  stop:0 亮色,
  stop:0.5 暗色,
  stop:1 亮色
);

/* 悬停时 */
background: qlineargradient(
  stop:0 亮色,
  stop:0.5 白色,  /* 金属高光 */
  stop:1 亮色
);
```

### 3. CRT屏幕效果
- 蓝/青色径向渐变
- 玻璃反光高光
- 水平扫描线
- 移动扫描线动画
- 真实CRT质感

## ?? 使用方法

### 启动
```bash
python main.py
```

### 测试功能

1. **全屏眼睛跟踪**:
   - 移动鼠标到屏幕任意位置
   - 观察吉祥物眼睛跟随

2. **生成Meme**:
   - 输入文字
   - 点击 [GEN] 按钮

3. **上传图片**:
   - 点击 [IMG] 按钮
   - 或拖拽图片到窗口

4. **随机生成**:
   - 点击 [RND] 按钮

5. **关闭程序**:
   - 点击 [X] 按钮
