# CRT Buddy v1.0.0 — Release Notes

Y2K Desktop Pet & Meme Generator for Windows.

---

## Download

- Binary: CRT_Buddy.exe (Windows 10/11 x64)
- Size: ~20–30 MB

---

## Run

1) Double-click CRT_Buddy.exe
2) Generate:
	- Text → click GENERATE MEME
	- Image → click UPLOAD IMAGE → apply effects
	- Random → click RANDOM Y2K EFFECT
3) Results are saved in `output/`

---

## Features

- CRT scanlines + RGB shift
- VHS jitter/ghosting/noise
- Holographic / Chrome / Neon glows
- Pixelate (8/16-bit)

---

## FAQ

Q: Missing DLL or cannot start?
A: Install Visual C++ Redistributable: https://aka.ms/vs/17/release/vc_redist.x64.exe

Q: Where are the generated images?
A: In the `output/` folder next to the EXE.

---

## Requirements

- Windows 10/11 (x64)
- Recommended RAM ≥ 4GB

# CRT Buddy v1.0.0 — Release Notes

Y2K Desktop Pet & Meme Generator for Windows.

---

## Download 下载

- Binary: CRT_Buddy.exe (Windows 10/11 x64)
- Size: ~20–30 MB

---

## Run 运行

1) 双击运行 CRT_Buddy.exe
2) 生成：
	- 文字 → 点击 GENERATE MEME
	- 图片 → 点击 UPLOAD IMAGE → 应用效果
	- 随机 → 点击 RANDOM Y2K EFFECT
3) 输出结果在 `output/` 目录

---

## Features 功能

- CRT 扫描线 + RGB 偏移
- VHS 抖动/拖影/噪点
- Holographic / Chrome / Neon 高光质感
- Pixelate 像素化（8/16-bit）

---

## FAQ 常见问题

Q: 运行提示缺少 DLL 或无法启动？
A: 安装 Visual C++ Redistributable: https://aka.ms/vs/17/release/vc_redist.x64.exe

Q: 生成后找不到图片？
A: 请在与 EXE 同级的 `output/` 目录查看。
